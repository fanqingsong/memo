### install
```
sudo apt-get install mysql-server
```

### start
```
sudo service mysql start
```

### login
```
mysql -u root -p
```

### open skip-grant-tables firstly
```
sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf
```
